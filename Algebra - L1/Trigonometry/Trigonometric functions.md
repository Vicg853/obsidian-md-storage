###### Negative $x$
- $cos(-x) = cos(x)$
- $sin(-x) = -sin(x)$
- $tan(-x) = -tan(x)$

###### Subtracting $\pi$
- $cos(\pi - x) = -cos(x)$
- $sin(\pi - x) = sin(x)$
- $tan(\pi - x) = -tan(x)$

###### Adding $\pi$
- $cos(\pi + x) = -cos(x)$
- $sin(\pi + x) = -sin(x)$
- $tan(\pi + x) = tan(x)$

###### Subtracting $\pi\over 2$
- $cos({\pi\over 2} - x) = sin(x)$
- $sin({\pi\over 2} - x) = cos(x)$
